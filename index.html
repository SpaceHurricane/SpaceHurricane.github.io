Space Hurricane - Not Your Mama's Rock Band

This accordian effect website template is released as an 'open source' design (under the Creative Commons Attribution 3.0 Licence), which means that you're free to use it for anything you want, providing you leave the 'design from HTML5webtemplates.co.uk' link in the footer.

This template uses a stand-alone accordian script from dezinerfolio. This script doesn't require any framework and is cross-browser compatible. Please be aware that with an accordian script, all of the content is in a single html page. Bear this in mind when you create your website, don't make it too big otherwise it will be slow to load.
